# application1
A clone of the productivity chrome app [Momentum](https://chrome.google.com/webstore/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca) on Vanilla JS

## Features
- [x] Weather with Geolocation
- [x] Clock
- [x] To Do List
- [x] Daily Photo
- [x] Offline Support
